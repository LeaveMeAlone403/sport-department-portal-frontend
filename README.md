# Sport Department Portal — Frontend

Клієнтська частина (Single Page Application) інформаційного порталу для Кафедри фізичної культури і спорту закладу вищої освіти. Забезпечує інтерактивний та адаптивний інтерфейс для користувачів.

## 🌐 Мережеві локальні адреси (Порти)
* **Фронтенд (React / Vite):** Running on `http://localhost:5173`
* **Бекенд (API, на який надсилаються запити):** Running on `http://localhost:5000`

## 🚀 Функціонал
* **Головний слайдер:** автоматична ротація актуальних подій.
* **Інформаційний блок:** відображення статистики та опису діяльності кафедри.
* **Плиточна галерея:** динамічна сітка зображень (Masonry/Grid).
* **Стрічка новин:** горизонтальна карусель із безкінечним прокручуванням.
* **Форма контактів:** валідація та відправка повідомлень через API + карта геолокації.
* **SPA Навігація:** плавний скрол до секцій із відстеженням через `IntersectionObserver`.

## 🛠 Стек технологій
* **Бібліотека:** React.js (Hooks, Refs)
* **Мова:** TypeScript (строга типізація)
* **Стилізація:** SCSS (змінні, nesting, міксіни), повна адаптивність
* **HTTP-клієнт:** Axios (асинхронний обмін даними)
